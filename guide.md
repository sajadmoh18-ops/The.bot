# دليل تشغيل بوت تلكرام (بايثون) 24/7 مجاناً لشخص مبتدئ 🤖

هذا الدليل راح يعلمك شلون تشغل البوت مالتك اللي يحتاج **Python** و **LibreOffice** (حتى يحول ملفات) على سيرفر مجاني ويبقى شغال طول الوقت بدون ما يطفي.

بما أن البوت مالتك يحتاج برنامج **LibreOffice**، فهذا معناه السيرفر لازم يكون قوي ويسمح لنا ننصب برامج إضافية. أفضل خيارين هم:
1. **Oracle Cloud (أوراكل كلاود):** يعطيك سيرفر حقيقي كامل ومجاني للأبد، بس يحتاج بطاقة مصرفية (فيزا أو ماستر) للتأكد من الهوية (ما يخصم فلوس).
2. **Render (ريندر):** أسهل بالاستخدام بس يحتاج نسوي شي اسمه "Docker" حتى ننصب LibreOffice.

---

## الخيار الأول: Oracle Cloud (الخيار الأفضل والدائم) 🏆

هذا الخيار يعطيك "حاسبة بالجو" (VPS) خاصة بيك، تكدر تسوي بيها أي شي.

### 1. شلون تسجل حساب
*   روح لموقع [Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/).
*   اضغط على **Start for free**.
*   اكتب إيميلك ومعلوماتك. **ملاحظة:** لازم تختار "المنطقة" (Home Region) قريبة عليك (مثل ألمانيا أو جدة) ولازم يكون عندك بطاقة مصرفية بيها مبلغ بسيط (تقريباً 1 دولار) بس حتى يتأكدون ويرجعوها فوراً.

### 2. شلون تسوي السيرفر (Instance)
*   بعد ما يفتح الحساب، دور على زر **Create VM Instance**.
*   بالإعدادات، اختار نظام التشغيل **Ubuntu 22.04**.
*   بالـ **Shape**، اختار النوع المجاني (غالباً يكون AMD أو Ampere ARM).
*   **مهم جداً:** حمل ملف الـ **Private Key** (ينتهي بـ `.key` أو `.pem`) واحفظه بحاسبتك، هذا هو "المفتاح" اللي يخليك تدخل للسيرفر.
*   اضغط **Create** وانتظر السيرفر يصير لونه أخضر (Running).

### 3. شلون ترفع الملفات وتدخل للسيرفر
*   استخدم برنامج اسمه **Termius** أو **PuTTY** على حاسبتك.
*   اكتب الـ **IP** مال السيرفر (تلكاه بصفحة السيرفر بأوراكل).
*   اسم المستخدم (Username) هو `ubuntu`.
*   استخدم "المفتاح" اللي حملته حتى تدخل.
*   حتى ترفع ملفات البوت، تكدر تستخدم برنامج **WinSCP** (سهل جداً، تسحب الملفات وتذبها بالسيرفر).

### 4. شلون تنصب المكتبات و LibreOffice
بعد ما تدخل للسوداء (الشاشة مالت السيرفر)، اكتب هاي الأوامر بالترتيب (انسخ والصق):

```bash
# تحديث السيرفر
sudo apt update && sudo apt upgrade -y

# تنصب بايثون و LibreOffice
sudo apt install python3 python3-pip libreoffice -y

# تنصب المكتبات اللي طلبتهن
pip3 install python-telegram-bot==22.7 openpyxl python-pptx Pillow fpdf2
```

### 5. شلون تشغل البوت 24/7
حتى البوت ما يطفي من تسد الشاشة، نستخدم برنامج اسمه `screen`:
*   اكتب `screen -S mybot`.
*   شغل البوت: `python3 main.py` (بمكان main.py اكتب اسم ملفك).
*   هسه اضغط من الكيبورد `Ctrl + A` وبعدين `D`.
*   مبروك! البوت هسه شغال بالخلفية. حتى لو سديت حاسبتك يبقى شغال.

## الخيار الثاني: Render.com (أسهل بس يحتاج ترتيب) ☁️

هذا الخيار أسهل لأن ما يحتاج فيزا، بس لازم ملفاتك تكون مرفوعة على موقع **GitHub**.

### 1. شلون تسجل حساب
*   روح لموقع [Render.com](https://render.com/).
*   سجل باستخدام حسابك مال **GitHub**.

### 2. شلون تسوي السيرفر
*   اضغط على **New** واختار **Web Service**.
*   اربط حسابك مال GitHub واختار "المستودع" (Repository) اللي بيه ملفات البوت.

### 3. شلون ترفع الملفات
*   ارفع كود البوت مالتك على GitHub.
*   **لازم** تضيف ملف اسمه `Dockerfile` بنص الملفات. هذا الملف هو اللي يگول لـ Render نصبلي LibreOffice.

### 4. محتوى ملف Dockerfile (انسخه وحطه بملف اسمه Dockerfile)
```dockerfile
FROM python:3.9-slim

# تنصيب LibreOffice والاعتمادات
RUN apt-get update && apt-get install -y libreoffice && apt-get clean

# نقل ملفاتك للسيرفر
WORKDIR /app
COPY . /app

# تنصيب المكتبات
RUN pip install python-telegram-bot==22.7 openpyxl python-pptx Pillow fpdf2

# تشغيل البوت
CMD ["python", "main.py"]
```

### 5. شلون تشغل البوت بشكل دائم
*   بموقع Render، من تختار الـ Dockerfile، هو راح يبني السيرفر ويشغله تلقائياً.
*   **ملاحظة:** ريندر المجاني "ينام" إذا ماكو طلبات (Requests). حتى تخليه كاعد 24 ساعة، تكدر تستخدم خدمة مجانية مثل [Cron-job.org](https://cron-job.org/) تنطي رابط البوت مالتك كل 10 دقائق حتى يبقى "كاعد".
